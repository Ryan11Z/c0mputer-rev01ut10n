Maths

 What effects have computers made in speeding up mathematical tasks that were previously slower and less efficient?

In the past century, many tasks have been sped up and become more viable from an economic standpoint. The most important example of this would be in mathematical calculations, which have seen the times to complete actions drop to a level in which these tasks can almost be taken for granted. This has hugely benefited human society as a whole, as completing these problems now takes far less time, enabling research to be conducted quicker, transactions to be completed in the blink of an eye and for air and spacecraft to travel with precision.
The basis by which computers work to calculate mathematical problems is centred on how the numbers are created, used and also how they interact with other numbers. The system used in computers to create numbers is known as binary. The method in which this is adding all of each individual bits like a switch. When creating whole numbers, with the smallest switch for a number (a whole number, so 1). If the number needs to be odd, the switch would then be switched 'on' by making that bit a 1. If it was switched off, the bit would be a zero. From there, each bit to the left becomes a larger power of 2, with the numbers after being 2, 4, 8, 16 etc. Eventually, this creates a more complex chain of bits, which ends up creating a number (e.g. 10011 would be 11 because switches for 8, 2 and 1 are 'on', with the total being 11.

Once a chain of bits is arranged in the desired fashion, it can then be used to help re-create and solve a problem. This is done by first assigning all the digits in order so than it can make sense as an equation, e.g. 4 times 5. Once this sequence is produced, you then use the software that it is necessary (in this case, a calculator) to enter this information. The data is proccessed through the software to create an answer based on how it is programmed (the answer being 20 in this instance).
